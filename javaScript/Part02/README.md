## Curso de Lógica de Programação

#### O que vou aprender?
```
- Desenvolver os conceitos básicos de lógica de programação por meio de atividades práticas
- Utilizar uma linguagem formal de programação no ambiente MS Visual Studio VB.NET
- Fornecer interfaces de entrada e de saída de informações e dados para serem processados 
  pelas instruções do programa
- Seguir instruções que utilizam expressões aritméticas e lógicas para manipular os dados do programa
- Escrever orientações que permitam controlar o fluxo das informações dentro do programa,
  por meio de estruturas de decisão e repetição

```

##### Variáveis:

```
- Operadores aritméticas;
- Operadores relacionais;
- Operadores Lógicos;
- Instruções Imperativos;
- Instruções condicionais;
- Lops de Programação;
- Vetores;
- Matrizes
```
