###  Constante e Variáveis:

```
- Valor constante ela se mantem inalterada em toda execusão do algoritimo;
--- Tipos:
         
                         
- Valor variável ela muda dependendo da situação  => São areas de memoria que se 
  armazenam valores que podem ser modificados durante a execução do algoritimo;
--- Tipos:
      
          Númericas => Inteiro (não tem casa décimas)
                    => Real (tem casa décimas)
          Alfanuméricas => Caracteres: (letras, númeres e caracteres especias )
          Lógicos   => Verdadeiro
                    => Falso
          Operadores Aritimétricos => +   Adição
                                      -   Menos ou subtração
                                      *   Multiplicação ou Asterisco
                                      /   Divisão
                                      n/o Resto
          Operadores Relacionais  =>  São operadores que estabelecem uma comparação entre duas variáveis ou 
          entre uma variável a uma constante. O valor dessa comparação é o resultado lógico.
                  os operadores são:
                       = igual;
                       <> diferente;
                       > maior que;
                       < menor que;
                       >= maior ou igual que;
                       <= menor ou igual que;
                       concatenar (colocar junto ou juntar )
                       
          Um pouco de  ( Algebra boleana )
          Expressão lógicas      =>
                   os lógicos são:
                       E:   Intersecção (O que é comum as dois)
                                 Se pelo menos uma condição for falso retorna tudo falso
                       OU:  União (E o conjunto)
                                 Se pelo menos uma condição fo verdadeiro retorna verdadeiro
                       Não: Negação (O que está fora)
                                Inverte o resultado
          funções:
                  Comndos de Decisão
                           comando SE (IF): Faz uma pergunta sobre o resultado de uma expressão relacional ou lógica. 
                                            O resultado de uma pergunta será sempre um valor lógico
                                            
                                            Ex: 
                                            SE(a>b) : Expressão Relacionais
                                            SE(a>b) e (c<d) : Expressão lógica
                                            
                            Comando de bloco: abertura(SE) e fechamento(Fimse)
          -----------------------------------------------------------------------------------------------------------------------------------
          Explicações de alguns pontos:          
                - todos os números em reais ele é guardado antes e depois da vírgula;
                - Não conseguimos fazer conta com valor caracteres
                
                - Expressão relacional ela tem uma conta de cada lado, para que possamos relacionar;
```
